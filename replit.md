# Premiere Cut Barbershop

## Overview
A premium barbershop website built with React, TypeScript, and Vite. Features a modern design with Tailwind CSS styling.

## Project Structure
- `App.tsx` - Main React application component
- `index.tsx` - React entry point
- `index.html` - HTML template with Tailwind config and fonts
- `components/` - React components
- `constants.ts` - Application constants
- `types.ts` - TypeScript type definitions
- `vite.config.ts` - Vite bundler configuration

## Tech Stack
- React 18
- TypeScript
- Vite
- Tailwind CSS (via CDN)
- Google Generative AI integration (@google/genai)

## Development
- Run `npm run dev` to start the development server on port 5000
- Run `npm run build` to create production build in `dist/`

## Environment Variables
- `GEMINI_API_KEY` - Google Gemini API key (optional, for AI features)
